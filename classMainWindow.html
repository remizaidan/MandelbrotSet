Detailed Description

The MainWindow class.

Manages the Main GUI. It is composed of two main sections:

  • A view widget to display the generated Mandelbrot Set.
  • An interactive section to change the settings

double  xmax,
double  ymin,
double  ymax 
)
privateslot

Respond to the updateNeeded() signal: starts updating the mandelbrot set.

The canculation of the Mandelbrot set happens in a concurent thread, leaving the GUI active with these changes:

  • Disable the save button and menu item.
  • Change the "Update" button into a "Stop" button, allowing to abort the calculation at any time.
Parameters
xminStart of the x coordinate range in the complex plane.
xmaxEnd of the x coordinate range in the complex plane.
yminStart of the y coordinate range in the complex plane.
ymaxEnd of the y coordinate range in the complex plane.

◆ doneUpdate

void MainWindow::doneUpdate ( )
privateslot

Respond to the pixmapReady() signal: update the GUI when a new set is ready.

When the calculation of the Mandelbrot set is done, resets the GUI to its original state:

  • Enable the save button and menu item.
  • Change the "Stop" button into an "Update" button, allowing to start a new calculation.
